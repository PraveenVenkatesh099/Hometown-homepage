# 🌍 Torshov Hometown Homepage

A simple and visually appealing landing page showcasing the beautiful district of **Torshov, Oslo**. This project was built using **HTML5** and **CSS3** to practice layout design, Flexbox, and styling.

## 📌 Project Preview

This webpage includes:

- 🏞️ Hero section with a background image
- 🎭 Top three attractions in Torshov
- 👤 Local guide information card
- 🎨 Clean color palette and modern UI
- 📱 Simple responsive layout using Flexbox

---

## 🚀 Features

- Hero banner with background image
- Activity cards with circular images
- Guide/Profile section
- CSS Flexbox layout
- Custom color palette
- Clean and readable code structure
